# PU Connect – College Management Database (MySQL)

## Overview

PU Connect is a relational database management system developed using MySQL for managing various academic and administrative activities within a college.

The database integrates students, teachers, courses, assignments, attendance, clubs, events, messaging, fees, and results into a centralized system, reducing manual work and improving data consistency.
